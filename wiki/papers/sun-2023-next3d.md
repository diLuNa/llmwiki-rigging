## Summary
Next3D proposes Generative Texture-Rasterized Tri-planes built on EG3D: neural textures are learned on a parametric mesh template (3DMM), then rasterized into three orthogonal feature planes for volume rendering. This couples fine-grained expression control from mesh-based methods with the topological flexibility of implicit volumetric rendering. A specialized mouth interior module handles the open-mouth region — previously unaddressed by mesh-based 3D GANs.

## Problem
EG3D-style 3D GANs lack expression control — they have no semantic deformation structure. Mesh-based 3D GANs have expression control but poor topological flexibility (can't handle hair, teeth). Next3D bridges both: mesh deformation provides controllable expression; neural texture rasterization into tri-planes provides volumetric quality.

## Method
**Generative Texture-Rasterized Tri-planes:** A 3DMM (FLAME-style) provides expression-controlled mesh geometry. Per-vertex neural texture features are generated by a StyleGAN2 backbone. These features are rasterized into three orthogonal 2D feature planes — creating expression-conditioned tri-planes compatible with EG3D's volume rendering pipeline.

**Mouth interior module:** A separate neural sub-network generates interior mouth geometry (teeth, tongue, inner lips) conditioned on jaw-open parameters — this region is not represented by the outer FLAME mesh.

**Training from 2D images:** The full system trains from unstructured 2D image collections with camera pose conditioning, no 3D supervision needed.

**Applications:** Expression animation (drive with FLAME coefficients), one-shot avatar creation (inversion), stylization (latent editing).

## Key Results
- State-of-the-art among 3D-aware head GANs with expression control.
- CVPR 2023 Highlight.
- Enables animation, one-shot avatar creation, and stylization in a single model.

## Limitations
- Generative model — requires GAN inversion to fit a specific identity (optimization-based, slow).
- Expression control via 3DMM coefficients, not per-vertex editing.
- Mouth interior module is a separate system; coupling can produce discontinuities.
